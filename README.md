# 🔍 GitHub Profile Finder

A clean and responsive web app to search any GitHub user's profile — built with pure HTML, CSS, and JavaScript. No frameworks, no libraries.

---

## ✨ Features

- 🔎 Search any GitHub username
- 👤 View profile photo, name, bio, and location
- 📊 See followers, following, repositories, and gists count
- 📁 Browse top 8 repositories with stars and forks
- 🔃 Sort repositories by Stars, Recent, or Forks
- 📋 Copy profile URL with one click
- ⚠️ Handles errors (user not found, API issues)
- 📱 Fully responsive (mobile friendly)

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub REST API
